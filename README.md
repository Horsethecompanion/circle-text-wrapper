# Circle Text Wrapper

A browser-based tool for wrapping text around a circular path — no sign-up, no server, everything runs locally.

**Live:** https://horsethecompanion.github.io/circle-text-wrapper/

## Usage

1. Open `index.html` in any browser
2. Enter your text
3. Adjust radius, font size, letter spacing and direction
4. Click **Copy SVG** to grab the SVG code

## Features

- Real-time preview on canvas
- Configurable radius, font size, letter spacing
- Clockwise / counter-clockwise direction
- Exports a clean SVG with embedded font
- Works fully offline
